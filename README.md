 Ansible_role_nagiosxi
Stands up a Nagios XI server on Ubuntu/CentOS

Warnings
The role puts SELinux into permissive mode and disabled firewalld

Requirements
None

Role Variables
See the example playbook below, as well as defaults/main.yml

None
